Effective field goal(endpoint)

                        
Returns leaders in effective field goal percentage per game. EFG is calculated to show what percentage a
                        two point shot is worth compared to a three point shot. More information can be found 
                        here.
                        

                    

                    

                        
GET

                        
http://127.0.0.1:8000/api/leaders/efg%

                    

                    

                        
                        

                    
 
                


                


                

                    

                        
True shooting(endpoint)

                        
Returns leaders in true shooting percentage per game. TS is calculated to show what percentage a
                            player scores on all shots taken. More information can be found 
                            here.
                            

                    

                    

                        
GET

                        
http://127.0.0.1:8000/api/leaders/ts%
